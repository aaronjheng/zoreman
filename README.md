# Zoreman

A [foreman](https://ddollar.github.io/foreman/) clone in Zig.

## Installation

### From Source

```shell
git clone https://github.com/aaronjheng/zoreman
cd zoreman
zig build -Doptimize=ReleaseSafe
```
